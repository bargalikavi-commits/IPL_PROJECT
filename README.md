# 🏏 IPL 2022 Capstone Project

## 📌 Project Overview

This project analyzes the Indian Premier League (IPL) 2022 dataset to uncover meaningful insights about team performance, player achievements, match outcomes, toss decisions, and venue statistics.

Using Python and data visualization libraries, the project explores various aspects of the tournament and answers important cricket-related questions through data analysis.

---

## 🎯 Objectives

- Analyze IPL 2022 match data.
- Identify the most successful teams.
- Study toss decision patterns.
- Compare toss winners and match winners.
- Analyze victory margins (runs vs wickets).
- Find top-performing batsmen and bowlers.
- Explore venue-wise statistics.
- Generate actionable insights from IPL data.

---

## 📂 Dataset

The dataset contains IPL 2022 match-level information, including:

- Match date
- Teams
- Toss winner
- Toss decision
- Match winner
- Winning margin
- Player of the Match
- Top scorer
- Best bowler
- Venue details
- And other match statistics

**File:** `IPL.csv`

## 📊 Analysis Performed

### 1. Team Performance Analysis
- Which team won the most matches?
- Team win distribution.

### 2. Toss Analysis
- Toss decision trends.
- Toss winner vs match winner comparison.

### 3. Match Result Analysis
- Wins by runs.
- Wins by wickets.
- Comparison of winning methods.

### 4. Player Performance Analysis
- Most Player of the Match awards.
- Top scorers.
- Best bowling performances.

### 5. Venue Analysis
- Venues hosting the most matches.
- Ground-wise match distribution.

### 6. Custom Insights
- Highest winning margin by runs.
- Highest individual score.
- Best bowling figures in the tournament.

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/IPL-Capstone-Project.git
```

2. Navigate to the project folder:

```bash
cd IPL-Capstone-Project
```

3. Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

4. Open Jupyter Notebook:

```bash
jupyter notebook
```

5. Run:

```bash
IPL_Capstone_Project.ipynb
```
